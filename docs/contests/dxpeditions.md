## About This Page

DXpeditions are manually curated from public announcements. Each entry includes a
4-character Maidenhead grid, which IONIS V20 uses to predict band conditions
from KI7MT (DN13) to the DXpedition location.

Predictions reflect current solar conditions (SFI and Kp) and update every 3
hours. They show the best decodable mode per band — what you can realistically
work given current propagation.

Past DXpeditions are filtered automatically by end date.

Want to add a DXpedition? Submit a PR to the
[dxpeditions.yaml](https://github.com/IONIS-AI/ionis-hamstats/blob/main/data/dxpeditions.yaml)
file — just callsign, entity, grid, dates, and modes.
